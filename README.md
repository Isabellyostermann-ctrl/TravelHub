1) Tema do Projeto
Desenvolvimento de um sistema de banco de dados para armazenamento e recomendação de dicas de viagem. A proposta é permitir que usuários compartilhem experiências e encontrem informações personalizadas com base em seu perfil, como orçamento, tipo de viagem e preferências.

2) Funcionalidade de Maior Valor
A principal funcionalidade é a recomendação de dicas personalizadas. O sistema utiliza o perfil do usuário para filtrar e exibir as informações mais relevantes, além de considerar avaliações de outros usuários para garantir maior confiabilidade nas recomendações.
